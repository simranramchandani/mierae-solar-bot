# ☀️ Mierae Solar AI Sales Machine

A high-converting, multilingual AI assistant designed to qualify leads for the **PM Surya Ghar Yojana** scheme. This bot handles initial greetings, qualifies homeowners, and handles common objections to book free site visits.

## 🛠️ Features

### 1. Multilingual Support (Language Fluency: 20%)
The bot is fully functional in three languages to maximize reach across diverse regions:
* **Hindi**: Targeted at Northern/Central India.
* **Telugu**: Targeted at Andhra Pradesh and Telangana.
* **Odia**: Targeted at Odisha.
> **Note:** Use the sidebar dropdown in the app to switch languages instantly.

## 📁 Project Structure
* `app.py`: The main UI and Sales Logic execution.
* `sales_logic.py`: The "Brain" containing the **PM Surya Ghar Yojana** dataset (₹78,000 subsidy, 300 units free power, etc.).
* `requirements.txt`: List of dependencies for cloud deployment.

---

## ⚙️ Local Installation

1. **Clone the repository:**
   ```bash
   git clone: https://github.com/simranramchandani/mierae-solar-bot/tree/main
   ```
2. **Install dependencies:**
   ```bash
   pip install streamlit
   ```
3. **Run the Sales Machine:**
   ```bash
   streamlit run app.py
   ```
